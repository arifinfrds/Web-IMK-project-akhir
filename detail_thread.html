You can do this with only subclassing UICollectionViewFlowLayout.


Here are the steps:


  1. Create a UICollectionViewFlowLayout subclass
  2. Override shouldInvalidateLayoutForBoundsChange: and return YES
  3. Override layoutAttributesForElementsInRect: and layoutAttributesForItemAtIndexPath:. Call the super implementation to get the standard FlowLayout values.
  4. Create a method that both the methods from step #3 call that looks changes the layout attributes of the items based on their distance away from the center of the collection view’s center. Use math to set the correct itemSize / transform / 3D transform for the items.

Here’s a super rough example that sort of does what you’re looking for, without the correct math.


@implementation MMLayout


(CGSize)itemSize {


return CGSizeMake(200.0f, 200.0f);


}
